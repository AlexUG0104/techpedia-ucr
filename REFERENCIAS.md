# Referencias del Proyecto

El desarrollo de este proyecto se basó en los siguientes recursos y documentación:

1. **Documentación Oficial de Vue.js 3**  
   - Base principal para la reactividad (`ref`, `computed`), ciclo de vida (`onMounted`) y manejo de componentes (`props`, `emits`).
   - [https://vuejs.org/guide/introduction.html](https://vuejs.org/guide/introduction.html)

2. **Documentación de Vite**  
   - Referencia para la creación del proyecto y configuración.
   - [https://vitejs.dev/guide/](https://vitejs.dev/guide/)

3. **MDN Web Docs (Mozilla Developer Network)**  
   - Guía esencial para el uso de la API Fetch para cargar el JSON, la API Audio para la reproducción de sonidos y mejores prácticas de CSS (CSS Variables, Grid, Flexbox).
   - [https://developer.mozilla.org/es/](https://developer.mozilla.org/es/)

4. **Recursos Multimedia**  
   - Las imágenes SVG y logos fueron referenciadas de repositorios de código abierto como Devicon (https://devicon.dev/).
   - *(Añadir referencias de donde se descargaron los audios descriptivos una vez agregados al proyecto)*

5. **Uso de IA como Apoyo**  
   - Se utilizaron asistentes de inteligencia artificial como guía metodológica, diseño de estructura de componentes, y mejores prácticas para CSS puro y Vue 3. La IA sirvió para acelerar el maquetado del código base de manera pedagógica para el curso IF7102.
