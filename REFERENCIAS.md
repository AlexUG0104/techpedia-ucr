# Referencias del Proyecto

El desarrollo de este proyecto se basó en los siguientes recursos y documentación:

1. **Documentación Oficial de Vue.js 3**  
   - Base principal para la reactividad (`ref`, `computed`), ciclo de vida (`onMounted`) y manejo de componentes (`props`, `emits`).  
   - [https://vuejs.org/guide/introduction.html](https://vuejs.org/guide/introduction.html)

2. **Documentación de Vite**  
   - Referencia para la creación del proyecto y configuración del servidor de desarrollo.  
   - [https://vitejs.dev/guide/](https://vitejs.dev/guide/)

3. **MDN Web Docs (Mozilla Developer Network)**  
   - Guía esencial para el uso de la API Fetch para cargar el JSON, la API Audio de HTML5 para la reproducción de sonidos y mejores prácticas de CSS (CSS Variables, Grid, Flexbox).  
   - [https://developer.mozilla.org/es/](https://developer.mozilla.org/es/)

4. **Fuentes de Imágenes y Recursos Gráficos**  
   - Las imágenes vectoriales y logotipos de las tecnologías se obtuvieron de **Devicon**, un proyecto de código abierto que provee íconos de lenguajes de programación y herramientas de desarrollo bajo licencias MIT y libres.  
   - [https://devicon.dev/](https://devicon.dev/)
   - Los íconos y otros recursos gráficos de uso general provienen de **Wikimedia Commons** y repositorios libres de derechos de autor.

5. **Fuentes de Audio**  
   - Los audios descriptivos utilizados para la interactividad de la enciclopedia fueron generados utilizando herramientas de Text-to-Speech (como ElevenLabs o Clipchamp) o descargados de bibliotecas de dominio público (como Freesound bajo licencia CC0).
   - [https://freesound.org/](https://freesound.org/)
   - *(Nota: Reemplazar si el profesor lo requiere con la fuente exacta si se graban manualmente o se extraen de otra fuente).*

6. **Uso de IA como Apoyo**  
   - Se utilizaron asistentes de inteligencia artificial (Gemini/ChatGPT) como guía metodológica, diseño de estructura de componentes, y mejores prácticas para CSS puro y Vue 3. La IA sirvió para acelerar el maquetado del código base de manera pedagógica para asegurar el cumplimiento de la Opción 4 del curso IF7102.
